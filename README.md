# Movie-recommendation-system
A full-stack movie recommendation application built with FastAPI and Streamlit, providing personalized movie suggestions using content-based filtering with TF-IDF and genre similarity. The system fetches movie metadata and posters from TMDB (The Movie Database) and displays recommendations in a clean, interactive UI.

Search Movies: Search by movie title using keyword autocomplete.

Movie Details: View full movie details including poster, overview, genres, release date, and backdrop.

Recommendations:

TF-IDF Similarity: Content-based recommendations using movie descriptions.

Genre-Based Suggestions: Movies with similar genres.

Home Feed: Browse trending, popular, top-rated, now-playing, and upcoming movies.

Responsive UI: Built with Streamlit for modern, interactive front-end.

FastAPI Backend: REST API serving movie data and recommendations.

Caching: Short-term caching for fast search and reduced API calls.
